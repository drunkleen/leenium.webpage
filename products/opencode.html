Leenium OpenCode

A terminal-first OpenCode theme using the same palette, contrast, and semantic roles as the rest of Leenium. It keeps the TUI clean, calm, and easy to read.

Easy setup

Copy the JSON theme into your OpenCode config theme directory.

Use
