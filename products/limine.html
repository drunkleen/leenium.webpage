Install

Clone and run the installer as root:

git clone https://github.com/drunkleen/leenium.limine.git && cd leenium.limine && sudo ./install.sh

Highlights

Boot branding

Uses the shared accent and help colors for the Limine interface text and branding line.

Terminal palette

Maps the exact Leenium neutral and accent colors into Limine's graphical terminal.

Simple install

Copies a ready-made `limine.conf` and updates the existing UKI name entry.
