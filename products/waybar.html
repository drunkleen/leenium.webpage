Leenium Waybar

A Waybar theme that brings the shared Leenium palette into a floating, Omarchy-like bar layout.

GitHub Repo Palette
Wayland Bar Desktop

Install

Clone the theme into your Waybar config directory and restart Waybar:

mv ~/.config/waybar ~/.config/waybar.backup && git clone https://github.com/drunkleen/leenium.waybar.git ~/.config/waybar && pkill waybar && waybar &

Highlights

Floating modules

Pills, borders, and spacing are tuned to feel closer to the rest of the desktop stack.

Shared palette

Uses the same background, accent, and text colors as the rest of Leenium.

Config pair

Designed to work with Waybar CSS and JSONC configuration together.
