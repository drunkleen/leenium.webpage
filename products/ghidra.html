Install

Open Ghidra and import the theme file from the project.

Edit > Theme > Import... > leenium.theme

Colors comments, constants, keywords, types, variables, and functions for readable analysis.

UI coverage

Styles headers, toolbars, filter fields, selections, and other application chrome.
